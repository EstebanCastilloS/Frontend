Rutas

Las rutas en HTML son las direcciones que utilizamos para enlazar a otros recursos, como páginas web, imágenes, archivos, etc. Existen dos tipos principales de rutas: absolutas y relativas.

Rutas Absolutas

Una ruta absoluta es una dirección completa que incluye el protocolo (http:// o https://), el dominio y la ruta del recurso. Por ejemplo:

https://www.ejemplo.com/carpeta/archivo.html

Las rutas absolutas son útiles cuando enlazamos a recursos externos o cuando queremos asegurarnos de que el enlace funcione sin importar desde dónde se acceda.

Rutas Relativas

Una ruta relativa es una dirección que se basa en la ubicación del archivo actual. No incluye el dominio ni el protocolo. Por ejemplo:

carpeta/archivo.html

Las rutas relativas son útiles para enlazar a recursos dentro del mismo sitio web. Hay varios tipos de rutas relativas:

  • Ruta relativa al mismo directorio: Si el archivo está en el mismo directorio, simplemente usamos el nombre del archivo. Ejemplo: archivo.html
  • Ruta relativa a un subdirectorio: Si el archivo está en un subdirectorio, usamos el nombre del subdirectorio seguido del nombre del archivo. Ejemplo: subcarpeta/archivo.html
  • Ruta relativa al directorio padre: Si queremos acceder a un archivo en el directorio padre, usamos dos puntos (..) para subir un nivel. Ejemplo: ../archivo.html
  • Ruta relativa a varios niveles arriba: Podemos usar múltiples conjuntos de dos puntos para subir varios niveles. Ejemplo: ../../archivo.html
