Para insertar una imagen en HTML, se utiliza la etiqueta <img>.

Ejemplo de imagen

Atributos importantes

  • src: Especifica la URL de la imagen.
  • alt: Proporciona un texto alternativo si la imagen no se puede mostrar.
  • width: Define el ancho de la imagen.
  • height: Define la altura de la imagen.
